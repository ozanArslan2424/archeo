Bu bir sosyal sorumluluk projesi ile ilgilidir.

Bu proje bir Vite + React uygulamasıdır.

1. Repo'yu clone'layın ve directory içerisine `cd` yapın.
2. Dependency'leri indirin: `npm install`
3. Uygulamayı dev modda çalıştırın: `npm run dev`
4. Uygulamayı prod modda çalıştırın: `npm run build && npm run preview`
